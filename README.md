\# 🏥 Hospital Performance Analysis Dashboard



\## 📌 Problem Statement



Hospitals generate large volumes of operational data, but identifying patient trends, resource utilization, and performance metrics is challenging.

This project analyzes hospital data to improve decision-making and efficiency.



\## 🛠️ Tools Used



\* Power BI

\* Excel / SQL



\## 📊 Dashboard Features



\* KPI cards (Total Patients, Admissions, Revenue)

\* Department-wise performance

\* Patient admission trends over time

\* Bed occupancy and resource utilization



\## 🔍 Key Insights



\* Peak admission periods identified

\* Certain departments handle majority of patients

\* Resource utilization varies significantly across departments
